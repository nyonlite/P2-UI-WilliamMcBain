We have made a 2D platformer with an inventory system and a dialogue system.
To test the save system, the save and load functions are available on the pause menu.

Controls:
Move - WASD
Jump - Spacebar
Talk - Enter
Pause - Escape

Known issues: 
- Visual issues with menu
- Items do not respawn when loading a save where they weren't collected from a save where they were.
